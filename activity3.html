Activity 3: Bitwise Operators

What are Bitwise Operators?

Bitwise operators are used to perform operations on binary numbers. Each number is first converted to binary, and then the operation is done **bit by bit** (0s and 1s).

Common Bitwise Operators

Operator Name Description
& AND Returns 1 if both bits are 1
| OR Returns 1 if at least one bit is 1
^ XOR Returns 1 if bits are different
~ NOT Flips all bits (1 to 0, 0 to 1)
<< Left Shift Shifts bits to the left (adds zeros)
>> Right Shift Shifts bits to the right

Example

Let's try this:

Number A = 5   → Binary: 0101  
Number B = 3   → Binary: 0011

A & B = 0101 & 0011 = 0001 → Result: 1
A | B = 0101 | 0011 = 0111 → Result: 7
A ^ B = 0101 ^ 0011 = 0110 → Result: 6
